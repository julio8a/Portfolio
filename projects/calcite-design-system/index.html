Calcite Design System

Calcite Design System is a collection of design and development resources for creating beautiful, easy to use, cohesive experiences across apps with minimal effort. It includes a UI kit, icons, color schemes, and a web component library with UI elements such as buttons, panels, accordions, alerts, and many more.

Component documentation

All components in the design system contain extensive documentation on:

  1. Usage - high level list items on usage for component.
  2. Sample (component variant interactive sandbox. Along with CodePen integration.
  3. Best practices, rendered examples along with things to do and avoid with components.
  4. Accessibility: Keyboard navigation and other best practices
  5. Writing and copy guidelines
  6. API Reference

Icon search

https://developers.arcgis.com/calcite-design-system/icons/

An UI app for finding the correct UI icon, illustration, or map symbology for application. The mini app has a robust searching/filtering capabilities utilizing icon metadata.

This metadata includes: related keywords, icon category, and release number

Libraries

  1. Calcite UI Icons: Functional SVG user interface icons built at 16, 24, and 32 pixels

  2. Meridian icons: Decorative SVG illustration built at 48 and 64 pixels. Used primarily for coach marks, empty states, etc. (not publicly available)

  3. Calcite point symbols: map symbol SVGs built at 13, 17, and 21 pixels
