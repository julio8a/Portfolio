Designed the template for Esri.com's industry pages, which consist of 65 industries, 11 verticals and over 400 individual web pages. The design was based on modules, which means that if an industry does not have a specific section, like videos, it's option to display. Design was based on simple UI elements and easy to use navigation for optimal user satisfaction.
